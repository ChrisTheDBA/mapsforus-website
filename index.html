Easy-to-create maps for everyone!
  • Automatically map your data
  • Your whole team can update the map

Use Cases

Hackathons

Quick Data Visualization

Features

  • Built on Google Spreadsheets
  • Map automatically updates when data is changed
  • Built-in geocoding (get coordinates from addresses)
  • Several choices of map tiles
  • Customizable map markers
  • Markers can cluster when map is zoomed out
  • Anyone on your team can update the map

Create Your Map!

1) Duplicate the Google Doc Template

  • Go to the template spreadsheet
  • File -> Make a Copy
  • Give it a name and hit "OK"
  • Make your spreadsheet public (so the map can see it):
    • File -> Publish Document
    • Leave "Entire Document" selected
    • Leave "Web page" selected
    • Hit "Publish"

2) Add Your Data

  • In the "Information" sheet:
    • For each row, fill out the "Customization" column
    • Tiles can be previewed here
  • In the "Points" sheet:
    • Fill out the title and description with the data you would like to see displayed with each point
    • Fill out addresses or latitude-longitude pairs
    • Choose an icon for each marker (here you can put the name of any Font Awesome icon)
    • Use the dropdown in the Marker Color cell to select the color of each marker
    • Provide a color for each marker icon (any HTML color name or hex code is fine)
    • If you would like your map to have layers (categories), assign each marker to a layer

3) Get Your Coordinates

(If you already have coordinates, skip this step.)

  • Click and drag to select the addresses and the empty latitude and longitude cells
  • Geocode -> Geocode Selected Cells (Address to Lat, Long)
  • Wait for the script to finish running

4) Make Your Map!

  • In your Google Spreadsheet, copy the part of the URL between "speadsheets/d/" and "/edit"
  • Paste it in the box below and hit "Go!"

5) Embed in Your Site

  • In your Google Spreadsheet, copy the part of the URL between "speadsheets/d/" and "/edit"
  • Paste it in the box below and hit "Get Code"

  • Now you can copy the below code and add it to the HTML of your website!
